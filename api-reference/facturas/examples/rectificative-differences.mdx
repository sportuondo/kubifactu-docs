## Cuerpo JSON de la petición

Esta factura rectifica y aplica una diferencia de importes a una factura anterior (`2024-000008`) de `100.00€` en la base imponible y de `21.00€` en el IVA aplicado.

* El campo `invoice_type_key` utiliza el valor `R1` (Factura Rectificativa (Error fundado en derecho y Art. 80 Uno Dos y Seis LIVA)).
* El campo `rectificative_type_key` tiene el valor `I`.
* **NO se incluyen** los campos `rectified_tax_base` y `rectified_tax_quota`.
* Se incluye la agrupación `rectified_invoices`.
